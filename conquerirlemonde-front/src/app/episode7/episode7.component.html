Episode 7 : Let's kill them all to dominate

 Download and install ansible 
      
        http://docs.ansible.com/ansible/intro_installation.html#installing-the-control-machine
Connect in your planet 
      
        ssh jarjar@{{this.identityService.identity.yourIp}}
Install pypy 
      
        docker run -it --rm -v /home/jarjar:/home/host-core sergeyzh/coreos-pypy
Link lib if you have error 
      
        cd /home/jarjar/pypy/bin ;ln -s /lib64/libncurses.so.5.9 libtinfo.so.5
Modify Host file 
      
        vi /etc/ansible/hosts
Add all planets in your file 
      
        fleet ansible_host=54.194.247.99 ansible_port=22 ansible_user=jarjar
        
        [planets]
        fleet
        
        [planets:vars]
        ansible_ssh_user=jarjar
        ansible_python_interpreter="/home/jarjar/pypy/bin/pypy"
List all planets 
      
        ansible -i hosts planets --list-hosts
Ping all planets 
      
         ansible planets -i hosts -m ping --ask-pass --sudo
View all docker 
      
         ansible all -i hosts -m shell -a 'docker ps' --ask-pass --sudo
Kill all docker 
      
        ansible fleet -i hosts -m shell -a 'docker stop #MY_TROOPER' --ask-pass --sudo
